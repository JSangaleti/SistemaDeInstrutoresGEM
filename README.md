# Sistema de Gestão de Instrutores e Alunos do GEM

Sistema desenvolvido para a disciplina de Projeto Integrador com o objetivo de melhorar o controle de instrutores, alunos, aulas e acompanhamento de progresso do Grupo de Estudos Musicais (GEM) da CCB.

## Visão geral

A proposta do sistema é centralizar informações acadêmicas e administrativas do GEM, permitindo:

- autenticação de usuários com perfis distintos;
- cadastro de alunos, instrutores e comuns congregações;
- registro de aulas e presenças;
- acompanhamento de histórico e progresso dos alunos;
- geração de relatórios gerenciais.

## Tecnologias utilizadas

- **Backend:** Java + Spring Boot
- **Frontend:** Flutter
- **Banco de dados:** PostgreSQL
- **Containerização:** Docker + Docker Compose
- **Segurança:** Spring Security e variáveis de ambiente
